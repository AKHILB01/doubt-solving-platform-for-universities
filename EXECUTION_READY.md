## 🚀 Quick Start (3 Steps)

### 1️⃣ Start MongoDB
```bash
# Windows
mongod.exe

# Mac
brew services start mongodb-community

# Linux
sudo systemctl start mongod
```

### 2️⃣ Start Backend (Terminal 1)
```bash
cd server
npm run dev
```

### 3️⃣ Start Frontend (Terminal 2)
```bash
cd client
npm start
```

**✅ Application is running at http://localhost:3000**
